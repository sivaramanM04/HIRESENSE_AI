# HireSense AI 🎯

HireSense AI is an AI-powered interview preparation platform built using Streamlit and Generative AI.

It helps users prepare for interviews through AI-driven practice, resume analysis, coding evaluation, and mock interviews.


## 🚀 Features

### 🧠 Technical Interview
- Role based technical questions
- Difficulty selection
- AI feedback
- Performance report

### 💼 HR Interview
- Behavioral questions
- Communication analysis
- Confidence evaluation

### 📄 Resume Analyzer
- Resume PDF analysis
- ATS score
- Skill gap detection
- Resume based interview questions

### 💻 Coding Assessment
- Multiple programming languages
- AI generated problems
- Code quality review
- Complexity analysis

### 🎤 Mock Interview
- Real interview simulation
- Technical + HR questions
- Final performance report


## 🛠 Tech Stack

- Python
- Streamlit
- SQLite
- Generative AI API
- Streamlit Ace Editor
- PDF Processing
